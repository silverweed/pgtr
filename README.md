Requires [An Iced CoffeeScript compiler](http://maxtaco.github.io/coffee-script/) to build.

Easiest way to get it up & running:

1. Install npm

```sh
sudo pacman -Sy npm
```

2. Install ICS

```sh
sudo npm install -g iced-coffee-script
```

3. Run the script

```sh
./watch.sh
```

4. Open the page in browser

```sh
firefox index.html
```
